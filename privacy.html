Any information we collect is not used in any malicious manner. If anything shown seems misleading, please contact us at 7luxorbot@proton.me.

Data Collection

The 7Luxor Bot collects the following data:

  • Guild IDs
  • Guild Names
  • Channel IDs
  • Role IDs
  • User IDs
  • Message Timestamps
  • Message IDs
  • Past Avatars
  • Nicknames and Usernames
  • Message content when a command is invoked (stored for a maximum of 14 days) or when arguments are passed for commands
  • Last deleted message content (stored for a maximum of 2 hours or less)
  • Last message edit history (stored for a maximum of 2 hours or less)
  • Last Emoji Reaction History (stored for a maximum of 2 hours or less)

Analytics and Performance Monitoring

We collect anonymous usage data and performance metrics to improve our service quality and user experience. This includes:

  • Command usage statistics and patterns
  • Bot performance metrics (response times, system resource usage)
  • Feature adoption rates
  • Error tracking and debugging information
  • User interaction patterns with bot features
  • Server growth and activity metrics
  • Security incident detection and prevention data
  • Bot configuration changes
  • Command success/failure rates

This data is processed and stored using the PostHog analytics platform and hosted on our servers in Dallas, Texas. We collect this data to:

  • Improve bot performance and reliability
  • Identify and fix bugs
  • Optimize popular features
  • Detect and prevent abuse
  • Make data-driven decisions about new features
  • Monitor system health and resource usage

The analytics data is:

  • Pseudonymized (no personally identifiable information)
  • Stored securely with restricted access
  • Retained for a maximum of 30 days
  • Never sold or shared with third parties
  • Used solely for improving our service

You can request information about what analytics data we store by contacting us at 7luxorbot@proton.me. While we cannot allow individual users to opt out of analytics collection (as it is essential for service operation and security), we minimize the data collected to only whatâ€™s necessary.

Usage of Data

When a command is invoked, we store the message content for a maximum of 14 days for debugging purposes. We also store up to 18 entries for edited messages and sniping messages that expire in two hours or less in volatile memory.

Additionally, Guild IDs, Channel IDs, Role IDs, User IDs, and Message IDs are stored so that our system can aggregate data.

Nickname, Username, and Avatar changes are logged to enable the "namehistory" and "avatarhistory" commands. Users can clear this data themselves at any time.

Guild name changes are logged to enable the "gnames" command, and server administrators can clear this data themselves at any time.

Information Sharing

We do not sell or expose your information to others, nor do we share it with any third parties.

Data Removal

If you would like to request removal of your data from our active systems, please email us at 7luxorbot@proton.me. Please note that while we honor deletion requests for our active systems, we may be unable to remove your data from existing backups until they naturally expire according to our retention schedule. Response times may vary and could take up to 7 days.

Data Storage and Backups

Our primary data storage is located in Dallas, Texas. For redundancy and disaster recovery, we maintain replicated backups in data centers located in New York, USA, and Falkenstein, Germany.

Automated backups are performed every 8 hours (3 times daily) in the UTC timezone. These backups are retained for 7 days before being permanently deleted.
